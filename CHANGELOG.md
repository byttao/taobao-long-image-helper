# 更新日志

## v1.4.0 - 2026-07-10

- 新增任务列表双击复制图片文件到剪贴板。
- 双击完成任务后，可在任意文件夹中使用粘贴操作复制图片。
- 保留原有拖拽复制能力。

## v1.3.0 - 2026-07-10

- 初始化 GitHub 开源仓库结构。
- 添加 MIT 开源协议。
- 增加中文 README、版本说明和发布说明。
- 保持 GUI 工具功能：Chrome 路径配置、端口自动调整、保存目录设置、任务列表和拖拽复制。
- 保持页面操作随机等待策略，降低连续快速访问导致的误判风险。

## v1.2.0 - 2026-07-10

- 页面关键操作之间加入随机等待。
- 登录轮询和页面失败重试加入随机间隔。
- 更新发布包。

## v1.1.0 - 2026-07-10

- 添加应用图标。
- 底部增加版本号和日期。
- 修复任务列表向外部文件夹拖拽复制图片。

## v1.0.0 - 2026-07-10

- 初始 .NET 8 WinForms GUI 版本。
- 支持启动 Chrome、用户扫码登录、输入商品 ID 或链接获取长图。
- 支持商品名称搜索页定位图片，并在失败时使用商品详情页主图兜底。
