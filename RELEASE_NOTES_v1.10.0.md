# v1.10.0 发布说明

日期：2026-07-10

## 更新内容

- 移除商品详情页主图兜底逻辑。
- 店铺搜索页未命中目标商品时，任务直接标记失败，不再下载详情页主图。
- 失败任务会保留已获取的商品名称、sellerId、shopId。
- 如果已生成店铺搜索页排查链接，会写入“图片链接”列，可双击复制。
- 任务列表新增 sellerId 和 shopId 列，方便用户手动排查。

## 使用说明

1. 点击“初始化”，连接或启动 Chrome，并完成淘宝登录。
2. 输入商品 ID 或商品链接，点击“开始获取长图”。
3. 成功后双击“图片链接”列复制图片网址，双击其他列复制图片文件。
4. 失败后查看列表中的商品名称、sellerId、shopId 和“图片链接”列排查链接，必要时手动打开继续处理。
