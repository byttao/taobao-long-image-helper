# TaobaoLongImageHelper v1.8.0

发布日期：2026-07-10

## 本次发布

- 修复双击复制时出现的 STA 剪贴板异常。
- 任务列表新增“图片链接”列。
- 双击“图片链接”列时，复制图片链接网址。
- 双击其他列时，复制图片文件，可在外部文件夹粘贴。

## 使用说明

1. 获取长图任务完成后，查看任务列表。
2. 双击“图片链接”列，复制图片 URL。
3. 双击其他列，复制图片文件。
4. 到目标文件夹中粘贴，即可复制图片文件。

## 运行环境

- Windows
- .NET 8 Desktop Runtime
- Google Chrome
