# v1.11.0 发布说明

日期：2026-07-10

## 更新内容

- 保留原“自动获取”流程，新增“千牛源码辅助”流程。
- 新增千牛商品链接转换工具：输入商品ID或商品链接后，生成指定格式的千牛商品链接并复制到剪贴板。
- 支持粘贴商品页源码提取商品名称。
- 支持粘贴店铺页源码提取 sellerId 和 shopId。
- 手动源码流程仍然复用 Chrome 打开店铺搜索页定位并下载长图。
- 应用文件名改为中文：`淘宝商品长图提取工具.exe`。
- 发布压缩包改为带版本号：`淘宝商品长图提取工具-v1.11.0-win-x64.zip`。
- 压缩包内会先套一层 `淘宝商品长图提取工具` 文件夹。

## 使用说明

1. 点击“初始化”，连接或启动 Chrome，并完成淘宝登录。
2. 自动流程：在“自动获取”页签输入商品 ID 或链接，点击“开始获取长图”。
3. 手动流程：在“千牛源码辅助”页签输入商品 ID 或链接，点击“转换并复制”，到千牛端打开该链接。
4. 将千牛商品页源码粘贴到“商品页源码”，提取商品名称。
5. 将店铺页源码粘贴到“店铺页源码”，提取 sellerId/shopId。
6. 点击“按源码信息获取长图”。
