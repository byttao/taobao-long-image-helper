# 更新日志

## v1.7.0 - 2026-07-10

- 启动新 Chrome 后增加调试端口就绪检查。
- 只有端口确认可连接后，界面才提示初始化成功。
- 不同端口使用不同 Chrome profile 目录，减少已有 Chrome 实例吞掉启动参数的问题。

## v1.6.0 - 2026-07-10

- 初始化时优先连接当前端口上的已有 Chrome。
- 如果已有 Chrome 可连接，则直接在该浏览器中打开淘宝登录页。
- 只有当前端口不可连接时，才自动寻找可用端口并启动新的 Chrome。

## v1.5.0 - 2026-07-10

- 优先从 `span[class^="mainTitle"]` 的第一个匹配元素提取商品标题。
- 标题读取顺序为 `title` 属性、`value` 属性、文本内容。

## v1.4.0 - 2026-07-10

- 新增任务列表双击复制图片文件到剪贴板。
- 双击完成任务后，可在任意文件夹中使用粘贴操作复制图片。
- 保留原有拖拽复制能力。

## v1.3.0 - 2026-07-10

- 初始化 GitHub 开源仓库结构。
- 添加 MIT 开源协议。
- 增加中文 README、版本说明和发布说明。
- 保持 GUI 工具功能：Chrome 路径配置、端口自动调整、保存目录设置、任务列表和拖拽复制。
- 保持页面操作随机等待策略，降低连续快速访问导致的误判风险。

## v1.2.0 - 2026-07-10

- 页面关键操作之间加入随机等待。
- 登录轮询和页面失败重试加入随机间隔。
- 更新发布包。

## v1.1.0 - 2026-07-10

- 添加应用图标。
- 底部增加版本号和日期。
- 修复任务列表向外部文件夹拖拽复制图片。

## v1.0.0 - 2026-07-10

- 初始 .NET 8 WinForms GUI 版本。
- 支持启动 Chrome、用户扫码登录、输入商品 ID 或链接获取长图。
- 支持商品名称搜索页定位图片，并在失败时使用商品详情页主图兜底。
