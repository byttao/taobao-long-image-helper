# v1.9.0 发布说明

日期：2026-07-10

## 更新内容

- 调整任务列表默认列宽，所有列按比例显示，默认不再出现横向滚动条。
- 获取失败时，如果程序已经生成店铺搜索页排查链接，会写入“图片链接”列。
- 双击“图片链接”列可以复制成功图片链接或失败排查链接。
- 双击其他列继续复制图片文件，便于到文件夹中粘贴。
- 更新界面底部操作说明。

## 使用说明

1. 点击“初始化”，连接或启动 Chrome，并完成淘宝登录。
2. 输入商品 ID 或商品链接，点击“开始获取长图”。
3. 成功后双击“图片链接”列复制网址，双击其他列复制图片文件。
4. 失败后如“图片链接”列有内容，可双击复制该排查链接自行打开检查。
