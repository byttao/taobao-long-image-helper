# TaobaoLongImageHelper v1.4.0

发布日期：2026-07-10

## 本次发布

- 新增任务列表双击复制功能。
- 双击已完成任务行后，会把对应图片文件放入 Windows 剪贴板。
- 用户可在任意文件夹中使用 Ctrl+V 或右键粘贴复制图片。
- 保留原来的任务行拖拽复制能力。

## 使用说明

1. 获取长图任务完成后，在任务列表中双击该记录。
2. 程序提示/日志显示图片已复制到剪贴板。
3. 打开目标文件夹。
4. 执行粘贴操作。

## 运行环境

- Windows
- .NET 8 Desktop Runtime
- Google Chrome
