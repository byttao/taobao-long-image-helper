# v1.12.0 发布说明

日期：2026-07-10

## 更新内容

- 优化“自动获取”页签排版，默认界面更紧凑。
- 切换到“千牛源码辅助”页签时自动展开高度，切回“自动获取”时自动收起高度。
- 千牛源码辅助流程改为“一键提取并获取长图”，点击后自动提取商品名称、sellerId 和 shopId，再继续获取图片。
- 商品页源码和店铺页源码输入框改为 RichTextBox，支持粘贴更长源码。
- 商品名称、sellerId、shopId 改为结果显示框，不再手动编辑。
- 初始化时如果默认端口 9222 被旧 Chrome 占用，会保留 9222，不自动切换到其他端口。

## 使用说明

1. 点击“初始化”，连接或启动 Chrome，并完成淘宝登录。
2. 自动流程：在“自动获取”页签输入商品 ID 或链接，点击“开始获取长图”。
3. 手动流程：在“千牛源码辅助”页签输入商品 ID 或链接，点击“转换并复制”，到千牛端打开该链接。
4. 粘贴商品页源码和店铺页源码，点击“一键提取并获取长图”。
